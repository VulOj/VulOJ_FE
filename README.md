## Requirements

1. react, react-dom
2. react-router-dom
3. antd
4. sass
5. axios

## Install dependencies
```
> $ cd vul-oj-fe
```

```
> $ npm install
```

## Start the project

```
> $ npm start
```

## Build the project
```
> $ npm run build
```
